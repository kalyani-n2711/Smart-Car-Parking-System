# Smart Car Parking System using IoT

This project implements a Smart Car Parking System using Arduino and IoT concepts
to detect parking slot availability and automate gate control.

## Features
- Ultrasonic sensor-based vehicle detection
- Automatic gate control using servo motor
- Real-time slot status display on LCD
- LED indicators for parking availability

## Hardware Used
- Arduino Uno
- Ultrasonic Sensors (HC-SR04)
- Servo Motor (SG90)
- 16x2 LCD Display
- LEDs and Resistors
